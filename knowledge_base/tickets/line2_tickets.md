# Примеры обращений - 2-я линия поддержки

## Тикет #101
**Тематика:** Системная проблема
**Критичность:** Высокая (P2)
**Описание:** Не работает почтовый сервер для отдела продаж. Пользователи не могут отправлять письма.
**Эскалация с:** 1-я линия (после 2 часов без решения)
**Решение:** Обнаружена проблема с конфигурацией почтового сервера. Конфигурация исправлена, сервис восстановлен.
**Время решения:** 3 часа
