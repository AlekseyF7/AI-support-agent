# Быстрое решение ошибки 403 (регион не поддерживается)

## Проблема

Ошибка: `Error code: 403 - unsupported_country_region_territory`

OpenAI API недоступен в вашем регионе.

## Быстрое решение

### Шаг 1: Добавьте прокси в `.env`

Откройте файл `.env` и добавьте строку:

```env
OPENAI_PROXY=http://your-proxy-server:port
```

**Примеры:**
```env
# HTTP прокси
OPENAI_PROXY=http://proxy.example.com:8080

# HTTPS прокси
OPENAI_PROXY=https://proxy.example.com:8080

# С аутентификацией
OPENAI_PROXY=http://username:password@proxy.example.com:8080

# SOCKS5 прокси
OPENAI_PROXY=socks5://proxy.example.com:1080
```

### Шаг 2: Перезапустите бота

```powershell
.\stop_bot.ps1
.\start_bot.ps1
```

## Где взять прокси?

### Варианты:
1. **VPN с прокси** - многие VPN сервисы предоставляют прокси
2. **Прокси-провайдеры** - специализированные сервисы
3. **Облачный сервер** - запустите бота на VPS в поддерживаемом регионе

### Бесплатные варианты (не для продакшена):
- Публичные прокси-списки (ненадежно, небезопасно)

## Альтернатива: VPN (самый простой вариант!)

Вместо прокси можно использовать VPN - это проще и не требует настройки кода:

1. **Установите VPN** (NordVPN, ExpressVPN, ProtonVPN и т.д.)
2. **Подключитесь к серверу** в США или Европе
3. **Запустите бота** - всё будет работать автоматически!

**Подробная инструкция:** См. `VPN_SETUP.md`

## Проверка

После настройки прокси при запуске бота вы должны увидеть:
```
[INFO] Используется прокси для OpenAI API
```

Если ошибка 403 исчезла - всё работает! ✅
